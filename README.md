# pre---Claude-Hackathon


AI Research Copilot (Claude + RAG)

Overview \
Tool to analyze text, documents, and notes
Turns unstructured input into structured insights
Built for fast iteration in a hackathon setting

Features \
Context retrieval using RAG
Claude-based reasoning over retrieved data
Structured outputs: insights, risks, action items
Chat mode and document analysis mode

Tech stack \
Frontend: Next.js or Streamlit
Backend: Python
LLM: Claude API
Vector database: Milvus
Embeddings: local or API

Pipeline \
Input text or document
Chunking and embeddings
Retrieve relevant context from Milvus
Claude processes query with context
Return structured output

Usage \
Paste text or upload document
Run analysis
View structured results

Notes \ 
Built under time constraints
Focused on core functionality
Designed for easy extension

Future work \
UI improvements
Multi-step workflows
Better document handling
External integrations
